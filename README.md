# Portfolio de Mathieu Pernot

Bienvenue sur mon portfolio en ligne ! Ce site présente mes projets personnels, mon expérience Erasmus ainsi que mon stage.

Le site est développé avec **Next.js 16** et utilise **Tailwind CSS** pour le style et **Framer Motion** pour les animations.

---

## Pages principales

- **Portfolio / Accueil** : Présentation générale et projets phares  
- **Erasmus** : Mon expérience Erasmus avec galerie d’images  
- **Stage** : Mon expérience professionnelle avec galerie d’images

---

## Technologies utilisées

- [Next.js](https://nextjs.org) – Framework React moderne  
- [React](https://reactjs.org) – Bibliothèque pour construire l’interface  
- [Tailwind CSS](https://tailwindcss.com) – Styling rapide et responsive  
- [Framer Motion](https://www.framer.com/motion/) – Animations et transitions  
- [Google Fonts](https://fonts.google.com/) – Polices optimisées
